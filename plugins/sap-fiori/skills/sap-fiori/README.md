# SAP Fiori Development Skill

Skill for SAP Fiori development — Fiori Elements page types, CDS UI annotations, SAPUI5 MVC apps, Fiori Launchpad configuration, Fiori Tools in BAS/VS Code, OData backend, navigation, and logistics app patterns.

## Skill Overview

- **Fiori Elements**: List Report, Object Page, ALP, Overview Page, Worklist — template-based apps
- **CDS Annotations**: @UI.lineItem, @UI.fieldGroup, @UI.selectionField, @UI.facets, @UI.chart
- **SAPUI5 MVC**: XML views, controllers, OData V2/V4 models, routing, fragments
- **Fiori Launchpad**: Tiles, target mappings, semantic objects, navigation parameters
- **Fiori Tools**: BAS generator, VS Code extensions, annotation modeler, preview, deployment
- **OData Backend**: SEGW (V2), CDS-based OData (V4), CAP, S/4HANA VDM APIs
- **Navigation**: Intent-based navigation, semantic objects, inner-app routing
- **Logistics Apps**: Delivery management, warehouse task UI, goods receipt, inventory, EWM/TM
