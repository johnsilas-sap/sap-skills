# SAP Mobile Development Kit (MDK) Skill

Skill for building SAP MDK mobile apps — metadata-driven pages, controls, OData bindings, JavaScript rules, offline store, barcode scanning, and deployment to SAP Mobile Services on BTP.

## Skill Overview

- **Project Structure**: Application.app, Pages, Actions, Rules, i18n, service config
- **Page Types**: ListReport, SectionedTable, FormCell, ObjectTable, ObjectHeader
- **Controls**: ObjectTableCell, SimplePropertyFormCell, SegmentedControl, Button, BarButton
- **Bindings**: OData property bindings, context bindings, expression bindings
- **Actions**: Navigation, OData CRUD, barcode scan, message/confirmation dialog, rules
- **Rules**: JavaScript context API — read, create, update, navigate, execute actions
- **Offline**: OfflineODataProvider, request objects, sync on demand
- **Barcode**: BarcodeServiceAction — scan Code128, QR, PDF417 from device camera
- **Deployment**: SAP Mobile Services cockpit, BAS deploy, MDK CLI
- **Logistics Apps**: Goods receipt, warehouse task confirmation, delivery check, inventory count
