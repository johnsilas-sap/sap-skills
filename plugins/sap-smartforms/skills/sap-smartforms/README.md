# SAP SmartForms Development Skill

ABAP development skill for SAP SmartForms — form building in SMARTFORMS, print programs, layout design, style sheets, conditions/events, PDF output, and migration to Adobe Forms.

## Skill Overview

- **Form Structure**: Global Settings, Form Interface (imports/globals/subroutines), Pages/Windows/Elements
- **Print Programs**: SSF_FUNCTION_MODULE_NAME → generated FM → SSFCTRLOP/SSFCOMPOP
- **Output Scenarios**: Spool printing, OTF retrieval, PDF conversion, email attachment
- **Layout Design**: Pages, windows (MAIN/secondary), text elements, tables, graphics, address nodes
- **Style Sheets**: SMARTSTYLES — paragraph formats, character formats, tab stops, page formats
- **Conditions**: Field conditions, ABAP expression conditions on pages/windows/table rows/elements
- **Events**: Table header/footer/line, page break events, alternative output
- **Migration**: SmartForms → Adobe Forms patterns, interface mapping, layout equivalents
- **EWM/TM Forms**: Delivery notes, warehouse labels, TO slips, DG declarations, packing lists
